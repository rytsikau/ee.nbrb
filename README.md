# ee.Nbrb

Данная 1С-обработка предназначено для отображения курсов обмена валют, установленных Национальным банком Беларуси. Доступна информация по всем валютам, начиная с 1991 года. Информация берется с сервера Национального банка.

Возможности программы:

* получение курсов по всем валютам на сегодня
* получение курсов по всем валютам на выбранную дату
* получение курсов по выбранной валюте за произвольную дату или период
* сохранение результата в текстовый или Excel файл

## Системные требования:
* платформа 1С 8.3
* доступ к Интернету
